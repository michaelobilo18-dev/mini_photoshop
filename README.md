## Description
This project is a simple desktop image filter application developed using Python.
The application allows users to load an image, apply basic image enhancement
operations such as grayscale conversion, blurring, edge detection, and save
the processed image.

The goal of the project is to demonstrate an understanding of image processing
concepts and desktop application development.

---

## Features
- Load image from local storage
- Convert image to grayscale
- Apply blur to image
- Perform edge detection
- Preview processed image
- Save edited image to disk

---

## Tools and Technologies
- Python
- Tkinter (for graphical user interface)
- NumPy
- Pillow (PIL) for image handling and display

---

## System Workflow
1. User launches the application.
2. User clicks the "Open Image" button to load an image.
3. The selected image is displayed in the application window.
4. User selects one of the available filters.
5. The processed image is displayed immediately.
6. User clicks the "Save Image" button to store the edited image.
7. Application is closed after use.
